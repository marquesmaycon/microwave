Projeto ao vivo: [microwave-chi.vercel.app](https://microwave-chi.vercel.app)

## Sobre

Simulador de microondas em React, com controles, programas pre-definidos e experiencia sonora.

O objetivo deste repositorio e demonstrar uma implementacao clara, organizada e facil de avaliar, com documentacao objetiva para recrutadores e pessoas tecnicas entenderem rapidamente o contexto do projeto.

## Funcionalidades

- Controles de tempo.
- Programas pre-definidos.
- Feedback sonoro.
- Interface interativa.
- Estados de simulacao.

## Stack

- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS**

## Como executar

```bash
git clone https://github.com/marquesmaycon/microwave.git
cd microwave
npm install
npm run dev
```

## Scripts disponiveis

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run preview`

## Destaques tecnicos

- Projeto criativo e diferenciado.
- Demonstra controle de estado e interacao.
- Boa peca visual para portfolio.
